  • Different types of structure tags
    1. Header tag = Used as a container for introductory content or a group of navigational links
      HEADER
    2. Footer tag = To define the footer section for its nearest ancestor sectioning content or the entire webpage
      FOOTER
    3. Section tag = To define a section in a document
      SECTION
    4. Article tag = To define an independent, self-contained piece of content
      ARTICLE
    5. Nav tag = To define a section of a page that links to other pages or to parts within the page
      NAVIGATION
    6. Main tag = To define the main content of a document
      MAIN
    7. Aside tag = To define content that is tangentially related to the content around it
      ASIDE
  • Div tag = It is the generic block container for flow content. It has no effect on the content or layout until styled in some way using CSS
  • Span tag = It is a generic inline container for phrasing content.
  • Block elements takes entire width of website
